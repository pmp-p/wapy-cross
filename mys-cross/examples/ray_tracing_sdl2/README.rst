Ray tracing
===========

A ray tracing example based on `rtiow Nelua`_.

Run these commands

.. code-block::

   $ mys build
   $ ./build/app > foo.ppm
   $ eog foo.ppm

and you should see

.. image:: https://github.com/mys-lang/mys/blob/main/examples/ray_tracing/output.png
.. _rtiow Nelua: https://github.com/nsauzede/realist/tree/master/rtiow/Nelua
